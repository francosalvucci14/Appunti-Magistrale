function Dense(in::Integer, out::Integer,
                σ = identity;
                initW = glorot_uniform,
                initb = zeros)
  return Dense(initW(out, in), initb(out), σ)
end

  W, b, σ = a.W, a.b, a.σ
  σ.(W*x .+ b)

using DifferentialEquations

function lotka_volterra!(du, u, p, t)
    α, β, δ, γ = p
    du[1] = α*u[1] - β*u[1]*u[2]
    du[2] = δ*u[1]*u[2] - γ*u[2]
end

sol = solve(prob)

function replicator!(du, u, fit_matrix, t)
    fitness = fit_matrix * u
    avg_fitness = u' * fitness
    du[:] = u .* (fitness .- avg_fitness)

Physics-informed neural networks (PINNs)

Scientific Models \(\approx\) Differential Equations \[ F = \frac {\mathbf p}{dt} \]

Experiments: info on \(\Delta x\) vs \(\Delta y\) for (unknown) law \(y = f(x)\)

💡 use ANN to improve Scientific Models

How to use ANN with ODEs?

Problem:

\[u' = f(u,t) \qquad t \in [0,1], u(0)=u_0\]

Assume \(NN(t) \approx u(t)\), then \(\frac {dNN(t)}{dt} = f(NN(t),t)\)

\[\text{loss}(p) = \sum_i \left(\frac{dNN(t_i)}{dt} - f(NN(t_i),t_i) \right)^2\]

Initial condition

How to satisfy the initial condition \(u(0)=u_0\)?

Solution 1

\[ \small \text{loss}(p) = (NN(0) - u_0)^2 + \sum_i \left(\frac{dNN(t_i)}{dt} - f(NN(t_i),t_i) \right)^2\]

Solution 2

Define the scaled ANN \(g(t) = t\cdot NN(t) + u_0\) so that \(g(0) = u_0\) and \[ \small \text{loss}(p) = \sum_i \left(\frac{dg(t_i)}{dt} - f(g(t_i),t_i) \right)^2\]

Let’s code it (v1)

Example: \[u' = \cos (2\pi t) , \qquad \text{ where } u(0) = 1.0\]

using Flux
NN = Chain(x->[x],
