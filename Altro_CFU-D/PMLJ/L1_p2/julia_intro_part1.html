Julia is homoiconic.

@macroexpand reveals macros:

@macroexpand @which 1+1
:(InteractiveUtils.which(+, (Base.typesof)(1, 1)))

Expressions and eval()

Meta.parse(s) let julia parse the string s and tell you the expression it represents.

Meta.parse("x+1")
:(x + 1)

dump shows the representation of a value.

dump(:(x+1))
Expr
  head: Symbol call
  args: Array{Any}((3,))
    1: Symbol +
    2: Symbol x
    3: Int64 1

eval

eval evaluates julia expressions.

eval(:(x+1))
24.140692632779267

Quiz

f1(x) = x + 1
function f2(x)
    eval(:(f1(x) = x))
    f1(x)
end
f2(1)
2
f2(1)
1
