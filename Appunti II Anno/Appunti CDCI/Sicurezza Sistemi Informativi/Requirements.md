Corso magistrale Sicurezza Sistemi Informativi

Strumenti utilizzati:

- Kali Linux (https://www.kali.org/) 
  Distro Linux Debian based con toolset per Offensive Security
 

- GNS3 (https://www.gns3.com/)
  Ambiente di virtualizzazione compatibile con Docker/VBox/VmWare/QEmu per costuire laboratori virtuali.


- Wireshark (https://www.wireshark.org/)
  Strumento per analizzare i protocolli di rete (sniffing).


- Metasploitable 2 (https://docs.rapid7.com/metasploit/metasploitable-2/)
  Server Ubuntu based intenzionalmente vulnerabile.


- Metasploit Framework (https://www.metasploit.com/)
  Framework per l'analisi e lo sfruttamento delle vulnerabilità.


- BurpSuite (https://portswigger.net/burp)
  Strumento per intercettare ed analizzare il traffico HTTP(s).
